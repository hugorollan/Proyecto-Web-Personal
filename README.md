# 🌐 Proyecto Web Personal - Hugo Rollán

Sitio web personal profesional desarrollado con HTML5 y CSS3, diseñado para presentar mi perfil como estudiante de Inteligencia de los Negocios, mis habilidades en análisis de datos y mi portafolio de proyectos académicos.

## 📋 Descripción

Este proyecto es una página web estática que sirve como portafolio profesional y carta de presentación digital. El sitio incluye información sobre mi formación académica, experiencia, habilidades técnicas y proyectos realizados durante mi carrera en la Universidad CEU San Pablo.

**Actividad 1 - Proyecto Web Personal**: Ejercicio práctico utilizando únicamente HTML y CSS para construir un sitio web personal completamente funcional y con diseño moderno.

## ✨ Características

- 🎨 **Diseño Moderno y Responsivo**: Interfaz limpia inspirada en diseños minimalistas profesionales
- 🚀 **Navegación Intuitiva**: Menú fijo con acceso rápido a todas las secciones
- 📱 **Adaptable**: Compatible con diferentes tamaños de pantalla
- 🎯 **Contenido Estructurado**: Información organizada de manera clara y profesional
- 🌈 **Efectos Visuales**: Transiciones y animaciones suaves para mejor experiencia de usuario
- ♿ **Accesibilidad**: Etiquetas semánticas y atributos aria para mejor accesibilidad

## 🚀 Instalación y Uso

### Requisitos Previos
- Un navegador web moderno (Chrome, Firefox, Safari, Edge)
- No se requieren dependencias adicionales

### Instrucciones

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/hugorollan/Proyecto-Web-Personal.git
   ```

2. **Navegar al directorio del proyecto**
   ```bash
   cd Proyecto-Web-Personal
   ```

3. **Abrir el sitio web**
   - Opción 1: Hacer doble clic en `index.html`
   - Opción 2: Abrir con un navegador específico
   - Opción 3: Usar un servidor local (recomendado para desarrollo):
     ```bash
     # Con Python 3
     python -m http.server 8000
     
     # Con PHP
     php -S localhost:8000
     
     # Con Node.js (usando http-server)
     npx http-server
     ```

4. **Acceder al sitio**
   - Si usas servidor local: `http://localhost:8000`
   - Si abres directamente: Se abrirá en tu navegador predeterminado

## 🎨 Personalización

Para personalizar el sitio con tu propia información:

1. **Contenido**: Edita los archivos HTML con tu información personal
2. **Estilos**: Modifica `css/styles.css` para cambiar colores, fuentes y diseño
3. **Imágenes**: Reemplaza las imágenes en la carpeta `images/` con las tuyas propias
4. **Enlaces**: Actualiza los enlaces a redes sociales con tus perfiles

### Paleta de Colores Principal:
- Primario: `#1d1d1f` (Negro suave)
- Secundario: `#86868b` (Gris medio)
- Fondo claro: `#f5f5f7`
- Acento: `#0071e3` (Azul)
